# برنامج مدبر الأرباح | Accountant Program

برنامج محاسبي متكامل مصمم خصيصاً لإدارة المشروعات الصغيرة (تجار جملة، عملاء، مخزن، وسيولة نقدية).

## المميزات الرئيسية
- **إدارة الموردين:** تتبع البضاعة الواردة، المبالغ المدفوعة، والديون المتبقية.
- **إدارة العملاء:** تتبع الطلبيات، ما تم تسليمه فعلياً، والمبالغ المستلمة.
- **المخزن الذكي:** حساب قيمة البضاعة المتوفرة حالياً بناءً على سعر الجملة.
- **السيولة النقدية:** حساب فوري للكاش المتوفر بناءً على (رأس المال + المقبوضات - المدفوعات).
- **واجهة متجاوبة:** تصميم يدعم العمل على الهواتف الذكية والأجهزة اللوحية بكفاءة.

## كيفية الرفع على GitHub
إذا كنت ترغب في رفع المشروع على مستودعك الخاص، استخدم الأوامر التالية في الـ Terminal:

```bash
git init
git remote add origin https://github.com/workspacesara53-del/accountant-program.git
git add .
git commit -m "إطلاق النسخة النهائية للمحاسب الذكي"
git branch -M main
git push -u origin main
```
