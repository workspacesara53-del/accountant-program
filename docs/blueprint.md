# **App Name**: مدبر الأرباح

## Core Features:

- نظام المصادقة الآمن: نظام تسجيل دخول وإدارة مستخدمين لضمان أمان البيانات وتقسيم الوصول للمستخدم الواحد، مع تخزين آمن لبيانات المستخدمين في قاعدة البيانات.
- إدخال طلبات العملاء: واجهة مستخدم بسيطة لإدخال تفاصيل طلبات العملاء يدوياً، بما في ذلك اسم العميل، المنتج، البائع، سعر الجملة، سعر البيع، والكمية المطلوبة، مع تخزين هذه البيانات في قاعدة البيانات.
- إدارة وعرض طلبات العملاء: قائمة عرض شاملة لجميع العملاء، وعند الضغط على عميل، يتم عرض تفاصيل جميع طلباته، حالة التسليم، الإجمالي، والمبلغ المستحق/المدفوع، مع جلب هذه البيانات من قاعدة البيانات.
- تتبع الموردين والمخزون: عرض قائمة البائعين/الموردين مع تفاصيل المنتجات المشتراة، أسعار الجملة الإجمالية، وحالة استلام كل منتج مع حقل لتسجيل الكمية المستلمة، مع إدارة وتخزين بيانات الموردين والمخزون في قاعدة البيانات.
- إدارة المعاملات المالية: ميزة إدخال الدفعات المسددة وتحديث تلقائي للمبالغ المستحقة للعملاء أو للموردين وحساب الرصيد الدائن والمدين، مع تخزين جميع تفاصيل المعاملات المالية بشكل دائم في قاعدة البيانات.
- لوحة التحكم المالية: شاشة رئيسية تعرض ملخصاً ديناميكياً للوضع المالي العام بما في ذلك الأرصدة المستحقة والمطلوبة، رأس المال، والأرباح الحالية، مع تجميع البيانات من قاعدة البيانات وعرضها.
- أداة الرؤى المالية المدعومة بالذكاء الاصطناعي: تحليل مبدئي لبيانات المعاملات المخزنة في قاعدة البيانات لتقديم ملخص سريع لأداء الأعمال وتحديد المجالات الرئيسية للأرباح والخسائر ورأس المال المتوفر باستخدام أداة توليد النص.

## Style Guidelines:

- Primary color: Deep, sophisticated blue-violet (#333399) to convey trust and professionalism. Hue 240, Saturation 50%, Lightness 40%.
- Background color: A very light, muted grey-blue (#F0F0F5) providing a clean, unobtrusive canvas. Hue 240, Saturation 20%, Lightness 95%.
- Accent color: A bright, clear sky-blue (#4DB8FF) for calls to action and highlighting key information, offering a vibrant contrast. Hue 210, Saturation 80%, Lightness 65%.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral aesthetic and excellent readability across different screen sizes, crucial for displaying financial data clearly.
- Use clear, minimalist line icons that communicate actions and categories effectively, aligning with a professional accounting tool (e.g., icons for 'add', 'edit', 'report', 'user', 'money').
- A structured and organized layout featuring a persistent left sidebar for main navigation (العملاء, الموردون, لوحة التحكم) and a main content area for detailed forms, lists, and data visualizations. Emphasize tabular displays for transaction clarity.
- Subtle, fast feedback animations on interactive elements like buttons and form submissions to confirm user actions without distracting from data review. Gentle transitions when navigating between different sections.
